## 中文版本

### 引言

本隐私政策适用于"AI辅助评价"Chrome浏览器扩展（以下简称"本扩展"）。我们高度重视您的隐私保护，并致力于确保您的数据安全。

### 数据收集声明

**我们郑重承诺：本扩展不收集、不传输、不存储任何用户数据。**

具体包括：

| 数据类型 | 是否收集 | 说明 |
|---------|---------|------|
| 个人身份信息 | ❌ 否 | 不收集姓名、邮箱、电话等任何个人信息 |
| API密钥 | ❌ 否 | 仅存储在用户本地浏览器中 |
| 浏览记录 | ❌ 否 | 不追踪用户访问的网站 |
| 评价内容 | ❌ 否 | 不保存用户生成的评价内容 |
| 使用统计 | ❌ 否 | 不上传任何使用数据 |

### 本地存储说明

本扩展仅在用户本地浏览器中存储以下配置信息：

- AI服务提供商设置
- API密钥（用户自行配置）
- 自定义标签和标签组
- 扩展启用/禁用状态

**所有这些数据都保存在您的本地设备上，不会上传到任何服务器。**

### 第三方服务

本扩展调用第三方AI服务（OpenAI或智谱清言）生成评价内容。请注意：

- 评价内容会发送至您选择的AI服务提供商
- 数据处理方式遵循相应服务商的隐私政策
- 我们不对第三方服务的数据处理负责

### 权限说明

本扩展请求的权限仅用于：

- **storage**：本地存储用户配置
- **activeTab**：在目标网站显示评价面板
- **host_permissions**：仅在 www.qidi-edu.com 上运行

### 数据安全

- 所有配置数据存储在Chrome浏览器的本地存储空间中
- 使用Chrome浏览器提供的安全机制保护数据
- 无后端服务器，无数据库，无数据传输

### 政策更新

我们保留更新本隐私政策的权利。更新后的政策将随扩展版本一同发布。

## English Version

### Introduction

This Privacy Policy applies to the "AI Feedback Assistant" Chrome browser extension (hereinafter referred to as "this extension"). We highly value your privacy protection and are committed to ensuring your data security.

### Data Collection Statement

**We solemnly pledge: This extension does not collect, transmit, or store any user data.**

Specifically:

| Data Type | Collected | Description |
|-----------|-----------|-------------|
| Personal identification information | ❌ No | No collection of names, emails, phone numbers, or any personal information |
| API keys | ❌ No | Stored only in user's local browser |
| Browsing history | ❌ No | No tracking of visited websites |
| Feedback content | ❌ No | No storage of generated feedback |
| Usage statistics | ❌ No | No upload of any usage data |

### Local Storage Explanation

This extension only stores the following configuration information in the user's local browser:

- AI service provider settings
- API keys (configured by user)
- Custom tags and tag groups
- Extension enabled/disabled status

**All of this data is saved on your local device and will not be uploaded to any server.**

### Third-Party Services

This extension calls third-party AI services (OpenAI or Zhipu AI) to generate feedback content. Please note:

- Feedback content is sent to your chosen AI service provider
- Data processing follows the respective service provider's privacy policy
- We are not responsible for third-party services' data processing

### Permissions Explanation

The permissions requested by this extension are used solely for:

- **storage**: Local storage of user configurations
- **activeTab**: Display feedback panel on target website
- **host_permissions**: Run only on www.qidi-edu.com

### Data Security

- All configuration data is stored in Chrome browser's local storage space
- Protected by Chrome browser's security mechanisms
- No backend servers, no databases, no data transmission

### Policy Updates

We reserve the right to update this Privacy Policy. The updated policy will be released with extension updates.

For any privacy-related questions, please contact us via:

- Send an email to: [Your Email]
